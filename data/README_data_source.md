# Data Source

## Dataset Name
Brazilian E-Commerce Public Dataset by Olist

## Source
Kaggle - Olist Brazilian E-Commerce Public Dataset

## Purpose of Dataset
This dataset is used to analyse e-commerce fulfilment, delivery performance, freight cost, customer satisfaction, seller performance and late delivery risk.

## Files Used
The raw dataset contains multiple CSV files, including:

- olist_customers_dataset.csv
- olist_geolocation_dataset.csv
- olist_order_items_dataset.csv
- olist_order_payments_dataset.csv
- olist_order_reviews_dataset.csv
- olist_orders_dataset.csv
- olist_products_dataset.csv
- olist_sellers_dataset.csv
- product_category_name_translation.csv

## Why Raw Data Is Not Uploaded
The raw CSV files are not uploaded to this GitHub repository because they are external dataset files. Users should download the dataset directly from Kaggle and place the CSV files inside:

data/raw/

## Data Use Notes
This project uses the dataset for learning, portfolio development and data analysis practice. Any conclusions are based on the dataset only and should not be overstated as representing all e-commerce markets.
